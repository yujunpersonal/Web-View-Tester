Identity Web View Tester

A developer tool for testing web authentication flows using WKWebView and ASWebAuthenticationSession on iOS.

Features

🌍

WKWebView Browser

Open any URL in a full WKWebView with back, forward, and reload navigation. See the current URL and page title in real time.

🔐

ASWebAuthenticationSession

Test OAuth and authentication flows using Apple's secure ASWebAuthenticationSession. Capture callback URLs and authorization codes.

🔗

Universal Link Handling

Listen for and capture universal links. View the full URL and parsed query parameters when a link is received.

📋

Result Capture

All results are displayed on the main screen with parsed parameters. Text is selectable for easy copying.
